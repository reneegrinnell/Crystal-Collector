Crystal Collector

A Blingy Math Game

Instructions:
A random number ("Target Value") wil appear at the start of the game.

Click on the four crystals below to add a specific point value to your score. Each crystal's value is hidden from you until you click it.

Win the game by matching your total score to the randomly selected target value. If the total score exceeds the target value, you lose!

The value of each crystal changes each time a new game starts.
